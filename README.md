<h1>My Portfolio Website!</h1>

## Objective
This website was created to store my photography and computer science projects in a single landing page. It also includes an updated resumé and contact page to keep business inquiries separate from my personal email. The domain is hosted on Cloudflare, forcing HTTPS before forwarding to GitHub Pages where the website is ran.

## Technology in Use
This site uses a few projects and APIs to work properly:
- [GitHub API](https://api.github.com/)
- [Formspree](https://https://formspree.io/)
- [nanogallery2](https://nanogallery2.nanostudio.org/)
- [jQuery](https://jquery.com/)
- Plus HTML, CSS, and Javascript, of course.
